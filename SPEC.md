# Fizz Letter - 泡沫来信

一个浪漫的匿名信网站。用户选择漂浮的气泡关键词，然后收到一封AI生成的手写风格信件。

## 核心流程

1. **打开网站** → 屏幕上浮着一组气泡，每个气泡里是一个模糊的、有文化暗示的词
2. **选择气泡** → 用户点击气泡，气泡"啵"地消失，被"揉"进手里
3. **三轮揉选** → 一共揉3轮，每轮从一组新气泡中选1-2个词。可以摇一摇换一组
4. **可选倾诉** → 揉完弹出对话框："想说点什么吗？" 用户可写可不写（textarea，placeholder："写下你想说的话..."）
5. **等待动画** → 几秒钟的信封飘落/展开动画
6. **收到信件** → 一封手写风格的信缓缓展开，内容温柔、带好感、微微暧昧

## 气泡关键词设计

关键词不直说风格，但自带文化暗示。用户凭直觉选。

## 信纸模板（CSS）

根据用户选的词的文化倾向，匹配对应信纸：

1. **羊皮纸风格** — 英伦/童话词汇 → 泛黄纸张，花体字，卷边
2. **和纸风格** — 日系词汇 → 浅色宣纸纹理，竖排或横排细字
3. **现代简约** — 韩系/美式 → 白底，无衬线字体，留白大
4. **复古信封** — 法式/中式 → 做旧信封，火漆印章，手写体

## 信的内容生成

### MVP版本（不需要API）：
- 预写30-50封信，按文化风格分类
- 根据用户选的词匹配最相关的一封
- 如果用户写了倾诉内容，在信里轻轻回应

### 未来版本：
- 调AI API（OpenRouter/Claude）实时生成
- prompt = 用户选的词 + 倾诉内容 + 风格指令

## 信的写作风格

- 语气温柔，像暗恋你的人写的
- 带一点暧昧但不过分
- 不假装是任何具体的人
- 像"一封来自远方的信"
- 150-300字
- 可以引用一句诗或歌词作为结尾

## 技术要求

- 纯前端：HTML + CSS + JavaScript
- 不需要后端（MVP）
- 移动端优先，响应式
- 气泡动画：CSS动画 + requestAnimationFrame
- 信件展开动画：CSS transition
- 字体：使用Google Fonts的手写体（Ma Shan Zheng中文手写 + Dancing Script英文）

## 视觉

- 背景：深色或渐变（深蓝→深紫），像夜空
- 气泡：半透明，微微发光，缓慢浮动
- 整体氛围：安静、浪漫、神秘
- 配色参考：#1a1a2e #16213e #0f3460 #e94560
